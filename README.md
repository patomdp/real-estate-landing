# Altavista Propiedades — Landing Page

Landing page para una empresa de real estate, construida como sitio estático (HTML + CSS puro, sin dependencias ni build step).

## Secciones

- **Hero** con llamados a la acción
- **Estadísticas** de la empresa
- **Propiedades destacadas** (venta, exclusiva, alquiler)
- **Servicios** (compra/venta, alquileres, inversiones)
- **Testimonios** de clientes
- **Formulario de contacto**

## Desarrollo local

Abrir `index.html` directamente en el navegador, o servir con:

```bash
npx serve .
```

## Deploy

Desplegado en Vercel como sitio estático.
